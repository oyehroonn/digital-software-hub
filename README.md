# DSM — Digital Software Market

Luxury software licensing concierge. The premium destination for genuine software licensing. Official Microsoft, Adobe & Autodesk partner.

## Tech stack

- **Vite** — Build tool
- **React** — UI library
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling
- **shadcn-ui** — Component library

## Getting started

### Prerequisites

- Node.js & npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Run locally

```sh
# Install dependencies
npm install

# Start development server (with hot reload)
npm run dev
```

The app will be available at `http://localhost:8080`.

### Build for production

```sh
npm run build
npm run preview   # Optional: preview production build locally
```

### Other scripts

- `npm run lint` — Run ESLint
- `npm run test` — Run tests
- `npm run test:watch` — Run tests in watch mode
